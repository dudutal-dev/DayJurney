# ✨ אור — מסע יומיומי

> מסע יומיומי של תפילה, התבוננות ולימוד  
> A daily spiritual journey through Psalms, Kabbalah and Torah

---

## על האפליקציה

**אור** היא אפליקציית web רוחנית עצמאית (standalone HTML), שמלווה את האדם במסע יומיומי דרך חיבור אישי בין מצבו הנפשי לבין תהילים, פרשת השבוע ומקורות אור מהמסורת.

המטרה: **מסע היזכרות באור**.

---

## שימוש

### Option 1 — GitHub Pages
```
Settings → Pages → Deploy from main branch → root
```
הגישה: `https://[username].github.io/or-daily/`

### Option 2 — פתיחה ישירה
פשוט פתח `index.html` בדפדפן. עובד offline לחלוטין.

### Option 3 — PWA Installation
ב-iOS: Safari → Share → "הוסף למסך הבית"  
ב-Android: Chrome → תפריט → "הוסף למסך הבית"

## טכנולוגיות

- **Vanilla HTML/CSS/JS** — ללא frameworks, ללא build process
- **PWA** — service worker, web manifest, offline support
- **localStorage** — שמירת נתונים מקומית
- **Google Fonts** — Frank Ruhl Libre + Noto Serif Hebrew
- **SVG** — עץ החיים ואייקון האפליקציה
